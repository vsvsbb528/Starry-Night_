Sistema de Riego Automatizado

Nuestro proyecto consiste en un sistema de riego inteligente que utiliza sensores de humedad para monitorear el estado del suelo y activar el riego solo cuando es necesario. Esto permite ahorrar agua, optimizar el crecimiento de las plantas y facilitar el cuidado de jardines y cultivos, todo incluyendo una aplicación de monitoreo fácil de usar, capaz de detectar las irregularidades del suelo fértil con facilidad.
